HTML - HyperText Markup Language


Every website is made using HTML.

HTML uses tags using angle brackets, > and <, to format text and create links and images.

In github, type the following to build a basic webpage:
<!DOCTYPE html>
<html>
<head>

<title>Hello!</title>

</head>
<body>

Hi, world!

</body>
</html>
 Then click the green commit button and reload the webpage to see the results!

Here are some more tags. Try them out and see what they do!
<b> (same as <strong>)
<i> (same as <em>)
<u>
<center>
<br>
<h1>
<h2>
<h3>
<h4>
<h5>
<a>
<img>
<br>

And on any webpage, you can press Command+Option+U to see the HTML code that makes it tick, for example how this kangaroo got here...



By itself, HTML is just boring black and white. To add color and style, we use CSS. You can add style to your webpage by adding information into the body tag, like this:
<body style="background-color:#FF9F00;font-family:impact;color:#2C2C2C">
