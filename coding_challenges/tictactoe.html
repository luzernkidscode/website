Tic-Tac-Toe

Let's make the simplest version of Tic-Tac-Toe you can think of. Once you have the basics working, you can continue to code the game, and make it fully functional and more fantastic.

Our simple version of Tic-Tac-Toe will have just one player, and that player will have 4 moves. At the end, the game won't even tell you whether you won or not, but it will show you the board. Sound simple? Then, once this is working, you can add more features to make your game more like regular Tic-Tac-Toe.

Here are some hints for how to write the code for your Tic-Tac-Toe game:
  • Use input() to let the player choose where to go. The player will enter a number 1-9 to select which spot they want to go.
  • Use 9 variables to store the spots on the board. Each spot will start out blank (spot1 = " ").
  • Use if to modify the variable that represents the spot where the user wants to make a move.
  • Print board with lines like this: print("{}|{}|{}".format(spot1, spot2, spot3))

Your game might look like this: 
Where do you want to go? 1
X| |
-----
 | |
-----
 | |

Where do you want to go? 5
X| |
-----
 |X|

 | |
